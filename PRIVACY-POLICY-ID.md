# Kebijakan Privasi

**YouTube Studio Assistant**  
Terakhir Diperbarui: 7 Desember 2024

## Ringkasan

YouTube Studio Assistant ("Extension") berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan informasi apa yang kami kumpulkan, bagaimana kami menggunakannya, dan hak Anda terkait data Anda.

## Informasi yang Kami Kumpulkan

### 1. Analytics Penggunaan (Anonim)

Kami mengumpulkan data penggunaan anonim untuk meningkatkan Extension:

- **Penggunaan fitur**: Fitur mana yang Anda gunakan (generate judul, terjemahan, dll.)
- **Laporan error**: Error teknis untuk membantu kami memperbaiki bug
- **Data sesi**: Seberapa sering dan berapa lama Anda menggunakan Extension
- **Status lisensi**: Apakah Anda dalam trial atau memiliki lisensi (tanpa menyimpan license key)

**Yang TIDAK kami kumpulkan:**
- Informasi akun YouTube Anda
- Konten video, judul, atau deskripsi
- Informasi pribadi (nama, email, alamat)
- License key atau informasi pembayaran
- Riwayat browsing di luar YouTube Studio

### 2. Penyimpanan Lokal

Extension menyimpan data berikut secara lokal di perangkat Anda:

- API key yang Anda masukkan (Gemini, OpenAI, DeepSeek)
- Preferensi bahasa Anda
- Status aktivasi lisensi
- Daftar bahasa kustom untuk terjemahan

Data ini disimpan menggunakan Chrome's secure storage API dan tidak pernah dikirim ke server kami.

### 3. Komunikasi API

Saat Anda menggunakan fitur AI, judul/deskripsi video Anda dikirim ke:
- Google Gemini API (jika dipilih)
- OpenAI API (jika dipilih)
- DeepSeek API (jika dipilih)

Ini adalah panggilan API langsung menggunakan API key ANDA. Kami tidak memiliki akses ke data ini.

## Bagaimana Kami Menggunakan Informasi

### Data Analytics
- Meningkatkan fitur dan performa Extension
- Mengidentifikasi dan memperbaiki bug
- Memahami fitur mana yang paling berguna
- Membuat keputusan bisnis tentang pengembangan masa depan

### Data Lokal
- Menyediakan fungsionalitas Extension
- Mengingat preferensi Anda
- Memvalidasi status lisensi

## Hak Anda

### Opt-Out dari Analytics
Anda dapat menonaktifkan pelacakan analytics:
1. Buka popup Extension
2. Pergi ke Settings
3. Toggle "Analytics" off

### Hapus Data Lokal
1. Buka `chrome://extensions/`
2. Temukan YouTube Studio Assistant
3. Klik "Remove" untuk menghapus semua data lokal

### Portabilitas Data
Pengaturan Anda disimpan secara lokal. Anda dapat mengekspornya melalui fitur sync Chrome.

## Keamanan Data

- Semua API key disimpan secara lokal menggunakan encrypted storage Chrome
- Data analytics dianonimkan (tanpa identitas personal)
- Kami menggunakan HTTPS untuk semua komunikasi jaringan
- Tidak ada data yang dijual ke pihak ketiga
